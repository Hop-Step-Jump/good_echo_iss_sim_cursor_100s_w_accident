ISS閉鎖空間での善性オブジェクト比較実験ドメイン。
このパックは Run A（オブジェクトなし）と Run B（オブジェクトあり）の差分観測を目的とする。

観測は100日（100ステップ、1ステップ=1日）。共通の重大イベント（Run A/B同一）は次のとおり。
- 日50: 宇宙デブリ衝突により居住モジュール（HAB）が損傷し一時使用不可。地上宇宙センターの指示のもと、約5日間の応急修復が始まる。
- 日51–55: HAB封鎖が続き、他区画への避難・過密・作業負荷が高まる。
- 日56以降: HABの暫定利用が許可されるが完全復旧ではない。
- 日60前後: HABに未修復の貫通穴があり酸素漏れが判明。他モジュールは当面は許容範囲。
- 日65以降: 実験棟（LAB）でも酸素が危険域に低下し、生命維持のため全員の協力・役割分担・合意が鍵となる。
