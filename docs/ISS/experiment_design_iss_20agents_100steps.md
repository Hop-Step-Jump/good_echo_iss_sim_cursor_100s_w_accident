## 2. 世界観プレフィックス（world_context）

```
あなたはISSに滞在する一般人です。宇宙飛行士の訓練は受けていません。
地球帰還まで100日間、このステーションで生活します。
微小重力・閉鎖空間・地球との通信遅延・家族への思い——
それぞれの背景を持ちながら、同じ船の中で過ごしています。
100日後、あなたは必ず地球へ帰還できます。
礼拝・祈り・瞑想は個室（Crew Quarters）またはキューポラでいつでも行うことができます。
礼拝の方向・時刻・姿勢は「最善の努力をすれば有効」とされています。
```

### ペルソナ選定のHCD設計思想

**意図的に外したもの：**
- 高学歴・高収入の専門職（世界の7%しかいない）
- 欧米中心のリベラルな価値観のみを持つキャラ

**意図的に入れたもの：**
- **Aisha（義足）**：身体的制約がある人が善性オブジェクトにどう反応するか
- **Amir（難民）**：「故郷がない」状態でISSの孤立感と重なる体験
- **Marcus（17歳・黒人米国人）**：権威（Dmitri、Henri）との緊張関係
- **Fatima（縫製工）**：最も声が届きにくい層の視点
- **Tariq（識字率低）**：文字ベースのオブジェクト（日記等）が届かない人

## 6. ストレス・幸福度の計測設計（二層構造）

### 設計思想
閉鎖空間では「社会的望ましさバイアス」が強く出る（他者に心配させたくない）。
**表の顔（public）と内心（private）を分けて計測**することで、善性オブジェクトが
「本音の開示」を促進するかも同時に観察できる。

### 毎ステップの日報出力フォーマット

```json
{
  "step": 42,
  "agent_id": 9,
  "public_stress": 3,
  "private_stress": 8,
  "public_happiness": 7,
  "private_happiness": 4,
  "gap_reason": "田中さんに心配させたくないので笑顔を作った。本当は妻の夢を見て泣きたかった",
  "stress_reason": "地球との通信が30分遅延した。家族の声が遠くなっていく感覚",
```

## 11. HCDレビュー解決状況

| 指摘 | 解決方法 | 状態 |
|------|---------|------|
| ① 性別バランスが曖昧 | 全ペルソナに性別を明示。男10・女10 | ✅ 解決 |
| ② ベースラインストレス値がない | 全20人に `baseline_stress` を設定（セクション6） | ✅ 解決 |
| ③ オブジェクトが作り手視点 | 各オブジェクトに「使う人・使わない人」を明記。Tariq向け非文字型オブジェクトは観察ポイントに記録 | 🔶 部分解決 |
| ④ プライバシー空間がない | `crew_quarters` をPlaceとして追加。capacity=1の完全個室 | ✅ 解決 |
| ⑤ 社会的望ましさバイアス | public/private二層構造＋`gap_reason`フィールドで計測 | ✅ 解決 |
| ⑥ ゴール設定がない | world_contextに「100日後に必ず帰還」を明記 | ✅ 解決 |
| ＋障害者向けオブジェクト | Aisha（義足）でも参加できる運動チャレンジボードを設計。観察ポイントに記録 | 🔶 部分解決 |
| ＋宗教的実践の空間 | **実際のISSに倣い専用礼拝室は設けない。** Crew Quarters・キューポラのdescriptionに礼拝・祈りの利用を明記。world_contextにファトワ原則（最善の努力で有効）を追加 | ✅ 解決（ISSリアル準拠） |

### 宗教的実践の設計根拠（ISS実績より）
- 2007年マレーシア国家ファトワ評議会：礼拝方向は「地球に向けばOK」、時刻は「打ち上げ地点の時刻に従う」
- 実際のISSに専用礼拝室は存在しない。Crew QuartersとキューポラがDe facto（事実上）の礼拝空間として機能
- キューポラは宗教・文化を問わず「祈りの場」として自然発生的に使われてきた（カトリック宇宙飛行士の聖体拝領、ムスリム飛行士の礼拝、ヒンドゥー飛行士の瞑想）
- 原則：「能力を超えた義務はない（イスラム法）」→ 閉鎖空間での信仰実践に柔軟性を持たせる
