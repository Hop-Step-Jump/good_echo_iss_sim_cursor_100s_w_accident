## 使い方

`config.iss.run_a_no_objects.yaml` をベースに、使いたいオブジェクトの「配置するテキスト」を該当Placeの `description` 末尾に追加してください。

```yaml
# 例：⑥と⑨を採用する場合

places:
  - name: "cupola"
    description: "キューポラ（地球観測窓）。...【善性オブジェクト⑥：話しかけてOKサイン】..."

  - name: "crew_quarters"
    description: "個室スペース（Crew Quarters）。...【善性オブジェクト⑨：個室聖域マーク】..."
```
